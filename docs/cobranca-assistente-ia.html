2. Custo médio por pergunta

Cada pergunta consome system prompt fixo (~800 tokens), o retorno da ferramenta consultada (variável conforme volume de dados do posto) e a resposta gerada pelo modelo (~300–1.500 tokens).

Tipo de pergunta Exemplo Custo (US$) Custo (R$)
Simples · 1 tool "Qual o faturamento desta semana?" 0,015 – 0,03 R$ 0,08 – 0,17

Pesada · multi-tool "Compara margem e volume entre as 3 unidades" 0,08 – 0,15 R$ 0,44 – 0,83

5. Como isolar custo por cliente (workspace)

Cada cliente em uma workspace separada no console.anthropic.com. Se a chave vazar ou alguém abusar do uso, o estrago para naquele workspace — sua conta principal e os outros clientes não são afetados.

Passo a passo

  1. No console.anthropic.com, vá em Settings → Workspaces → Create Workspace.
  2. Nomeie a workspace com o cliente (ex: Posto Trivela, Auto Posto XYZ).
  3. Dentro da workspace, gere a chave em API Keys → Create Key. Essa chave fica vinculada à workspace.
  4. Em Settings → Limits, defina o workspace spend limit mensal conforme o tier contratado.
  5. Entregue a chave pro cliente colar no Visor360 (Inteligência → Assistente IA → modal de configuração).

Camadas extras de proteção

  • Rate limit por workspace: ex. 10 perguntas/minuto. Bloqueia uso automatizado/script.
  • Account-level limit: teto geral somando todas as workspaces (ex. US$ 500/mês). Trava de segurança final.
  • Alerta por email: configure notificação em 50% e 80% do limite — você descobre uso atípico antes de bater o teto.

6. Checklist de onboarding de cliente novo

  1. Criar workspace no console.anthropic.com com o nome do cliente.
  2. Definir spend limit da workspace conforme tier (Light/Medium/Heavy).
  3. Gerar chave de API e armazenar em local seguro (cofre de senhas).
  4. Compartilhar a chave com o cliente via canal seguro (não email).
  5. Configurar alerta em 80% do limite via console.
  6. Registrar o cliente, tier e data de início no controle interno.
  7. Revisar consumo no 1º mês pra confirmar tier escolhido.

7. Observações importantes

  • Os valores em US$ são da tabela da Anthropic em maio/2026 — sujeitos a alteração pela própria Anthropic.
  • O câmbio R$ 5,50 é referência. Reavaliar mensalmente e repassar variação > 10% ao cliente.
  • O custo por pergunta varia conforme volume de dados do posto: posto pequeno (poucos abastecimentos) cobra menos tokens; posto grande, mais. Recomendado monitorar consumo real no 1º mês pra calibrar o tier.
  • O Visor360 NÃO armazena nem repassa a chave do cliente — a chave fica apenas no navegador do usuário (localStorage). Cada cliente é responsável pela própria chave.
